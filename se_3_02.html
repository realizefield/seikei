歴史的仮名遣いの読み方（発音のしかた）

現代文に使われている仮名遣いを「現代仮名遣い」といい、古文に使われている仮名遣いを「 歴史的仮名遣い 」といいます [cite: 4]。これは、 平安 時代中期ごろに実際に発音されていた形が基本になっています [cite: 4]。

読み方の決まり
古語辞典に載っている「こうい」などは「表記のしかた」を示しています 。一方で、テストなどで問われるのは「読み方（発音のしかた）」です 。書くときは、長音記号（ー）の代わりに「う」をあてはめると正しい現代仮名遣いの表記になります（例：シューク→しゅうく） 。

1) 語中・語末の「は行」: 単語の最初を除いた「は・ひ・ふ・へ・ほ」は、 ワ・イ・ウ・エ・オ と読みます [cite: 7, 8]。
例：にほひ → ニオイ [cite: 9]、いふ → イウ [cite: 44]、おもふ → オモウ。

2) ワ行の「ゐ・ゑ・を」: これらは、 イ・エ・オ と読みます [cite: 10]。
例：ゐなか → イナカ [cite: 30]、ゑひ → エイ [cite: 30]、をむな → オンナ [cite: 31]。

3) 「む」の読み方: 語中の「む」、助動詞「む」「らむ」「けむ」、助詞の「なむ」の「む」は ン と読みます [cite: 11, 12]。
例：らむ → ラン [cite: 54]、なむ → ナン。

4) 「ぢ・づ」: ジ・ズ と読みます [cite: 13]。
例：なんぢ → ナンジ [cite: 33]、ふみづき → フミズキ [cite: 142]、はづき → ハズキ [cite: 144]。

5) 「くわ・ぐわ」: カ・ガ と読みます [cite: 14]。
例：くわんぱく → カンパク [cite: 33]。

長音の決まり
母音に「ふ」がつく場合も、(1)と(6)の決まりにより長音で読みます [cite: 27]。nifudau（にふだう）は、決まりにより nifudau → niudau → nyūdō と変化し、 ニュードー と発音します [cite: 27, 29]。

6) 母音の連続: a・i・u・e・oが連続する場合は長音で読みます [cite: 15]。
しうく → シューク [cite: 16]、あふぎ → オーギ [cite: 31]、えうなし → ヨーナシ [cite: 32]。

7) 特定の組み合わせ: でうし → チョーシ [cite: 26]、かうい → コーイ [cite: 201]、翁（おうな） → オーナ [cite: 26]。
